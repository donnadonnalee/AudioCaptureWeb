# Audio Capture Pro 🎙️

ブラウザからシステム音（PCから出ている音）をキャプチャし、WebMまたはWAV形式で録音・保存できるウェブツールです。

Windowsの「ステレオミキサー」のような機能を、特別なソフトウェアのインストールなしにブラウザ（Vanilla JS & CSS）のみで実現しています。

## 🌟 特徴

- **システム音キャプチャ**: ブラウザの `getDisplayMedia` APIを利用して、高品質なシステム音を直接取得。
- **MP3形式出力**: `lamejs` を使用したMP3形式へのエンコード・保存（ビットレート選択可）。
- **無音自動分割**: 長い無音を検知して自動でトラックを分割。無音のみのトラックは自動破棄。
- **WAV形式出力**: 非圧縮のWAV形式（16bit PCM）への変換・保存に対応。
- **WebM形式出力**: 高圧縮・高音質なWebM形式での保存。
- **リアルタイム・ビジュアライザー**: Web Audio APIを用いた周波数スペクトル表示。
- **プライバシー配慮**: 全ての処理をブラウザ内で行い、音声データをサーバーに送信しません。

## 🚀 使い方

1.  `index.html` をブラウザで開きます。
2.  「保存形式」を選択（MP3を選択した場合はビットレートも選択可能）。
3.  必要に応じて「無音自動分割」を有効にし、秒数を設定。
4.  「画面共有を開始」ボタンをクリック。
5.  表示されるダイアログの下部にある **「システムオーディオを共有」** または **「タブの音声」** にチェックを入れて共有を開始。
6.  「録音開始」をクリックしてキャプチャ。
7.  終了後、ダウンロードリンクから保存。
